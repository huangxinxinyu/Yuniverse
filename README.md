# Yuniverse

Personal website for Yuniverse.

The deployable app lives in `frontend/`. Keep future site edits there only.

## Local Setup

```bash
cd frontend
npm install
npm run dev
```

## Common Commands

```bash
cd frontend
npm test
npm run build
npm run lint
```

## Deployment

Vercel is configured to build the `frontend/` app. The production domain is:
